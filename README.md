# Corrutinas (Android)

## Qué es
Proyecto de práctica para aprender y aplicar corrutinas en Android: concurrencia, suspensión, y ejecución en distintos dispatchers.

## Stack
- Kotlin
- Android Studio
- Kotlin Coroutines

## Features
- Ejemplos de `launch`, `async/await` y suspensión
- Uso de dispatchers (`Main`, `IO`, `Default`)
- Manejo de cancelación y errores (`try/catch`)
